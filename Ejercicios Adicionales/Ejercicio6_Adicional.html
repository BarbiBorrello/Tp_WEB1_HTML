El elemento HTML video

Para mostrar un video en HTML, use el elemento video.

Cómo funciona

El controlsatributo agrega controles de video, como reproducción, pausa y volumen. Es una buena idea incluir siempre los atributos widthy y height. Si no se configuran la altura y el ancho, la página puede parpadear mientras se carga el video. El elemento source le permite especificar archivos de video alternativos que el navegador puede elegir. El navegador utilizará el primer formato reconocido. El texto entre las etiquetas video y /video solo se mostrará en navegadores que no admitan el elemento video. Para iniciar un video automáticamente, use el atributo autoplay.
